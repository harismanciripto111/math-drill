# PLAN.md — Math Drill

## Goal
Website latihan matematika dasar yang membuat pengguna terbiasa dengan operasi aritmatika melalui sesi Pomodoro terstruktur.

## Core Features (DONE)
- [x] Pomodoro timer (25 menit kerja, 5 menit istirahat)
- [x] 4 operasi: penjumlahan, pengurangan, perkalian, pembagian
- [x] Pilihan lanjut/stop setelah break (opsional)
- [x] Streak system harian via localStorage
- [x] Statistik sesi (soal dijawab, benar, akurasi)
- [x] 3 level kesulitan (mudah 1-10, sedang 1-20, susah 1-50)
- [x] Animasi feedback benar/salah
- [x] Progress bar timer visual

## Future Enhancements
- [ ] Sound effects (Web Audio API)
- [ ] Mode sprint 60 detik
- [ ] Statistik mingguan/bulanan
- [ ] Notifikasi browser saat break selesai

## Stack
- Vanilla HTML/CSS/JS
- localStorage
- GitHub Pages
