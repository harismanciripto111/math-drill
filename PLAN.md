# PLAN.md — Math Drill

## Goal
Website latihan matematika dasar yang membuat pengguna terbiasa dengan operasi aritmatika (tambah, kurang, kali, bagi) melalui sesi latihan terstruktur dengan Pomodoro timer.

## Core Features
- [x] Pomodoro timer (25 menit kerja, 5 menit istirahat)
- [x] 4 operasi matematika (penjumlahan, pengurangan, perkalian, pembagian)
- [x] Pilihan lanjut/stop setelah break (opsional)
- [x] Streak system (harian, via localStorage)
- [x] Statistik sesi (soal dijawab, benar, akurasi)
- [x] 3 level kesulitan (mudah 1-10, sedang 1-20, susah 1-50)
- [x] Animasi feedback benar/salah
- [x] Progress bar timer

## Possible Future Enhancements
- [ ] Sound effects (ding saat benar, buzz saat salah)
- [ ] Leaderboard / high score per hari
- [ ] Mode tantangan (jawab sebanyak mungkin dalam 60 detik)
- [ ] Statistik mingguan/bulanan
- [ ] Mode gelap/terang toggle

## Stack
- Vanilla HTML/CSS/JS
